## Платформер для итогового проекта

Это рабочий прототип платформера на Pygame с логикой, коллизиями, камерой, предметами, врагом, здоровьем, победой и поражением. Графика собрана из квадратов, прямоугольников и кругов, чтобы потом было удобно заменить её на спрайты.

### Запуск

Просто запусти:

```powershell
python main.py
```

### Управление

- `A` / `D` или `Left` / `Right` - движение
- `Space` - прыжок
- `Esc` - выход

### Состояния игры

- Собери 4 предмета, чтобы появилась дверь финиша.
- Касание двери показывает `YOU WIN!` и останавливает игру.
- Если жизни заканчиваются, появляется `GAME OVER`.

### Структура

Основная логика находится в папке `my_platformer/`:

- `main.py` - запуск игры
- `config.py` - все константы уровня и баланса
- `camera.py` - камера, следующая за игроком
- `player.py` - игрок, движение, прыжок, здоровье
- `enemy.py` - враг со state machine
- `item.py` - предметы и дверь
- `world.py` - мир, платформы и проверки победы/поражения
- `utils.py` - HUD и текст
