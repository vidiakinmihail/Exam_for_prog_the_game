# Escape from The Garden 🎮

Финальный проект по программированию — полностью готовый пиксельный 2D-платформер, написанный на **Pygame** с использованием объектно-ориентированного подхода (ООП). 

В игре реализована полноценная физика коллизий, динамическая камера, плавающий многослойный параллакс-фон, враги с патрулированием, анимированные объекты, система сохраняемого статуса (HUD) и модульное тестирование.

## 🚀 Особенности проекта

* **Графика и Спрайты:** Полная интеграция пиксель-арта — анимированные персонажи, прорисованные тайлы платформ, интерактивные порталы, враги и монеты.
* **Живое Главное Меню:** Реализован анимированный задний план с эффектом многослойного параллакса (слои леса движутся с разной скоростью) и кастомными полупрозрачными кнопками.
* **Проработанный HUD (Интерфейс):** Геометрически выверенная панель статуса, отображающая текущий уровень, счетчик собранных монет и здоровье игрока в виде текстурированных сердечек.
* **Камера:** Плавное следование за персонажем с автоматическим ограничением по границам игрового мира.
* **Архитектура:** Чистое разделение логики (модульная структура), кэширование ресурсов (`@lru_cache`) для высокой производительности и стабильный FPS.
* **Автотесты:** Покрытие проекта Unit-тестами для проверки структуры уровней и валидности координат без необходимости ручного тестирования.

## 🕹️ Управление

* `A` / `D` или `←` / `→` — Движение персонажа
* `Space` (Пробел) — Прыжок
* `S` — Настройки (в главном или пауза-меню)
* `Esc` — Пауза / Выход в меню

## 📈 Игровой процесс и Уровни

Для прохождения игры необходимо собрать все монеты на уровне и добраться до магического портала:
* **Уровень 1:** Собрать 2 монеты 🪙 → Активация портала.
* **Уровень 2:** Собрать 4 монеты 🪙 → Активация портала.
* **Уровень 3:** Собрать 6 монет 🪙 → Финальный портал и победа (`GAME COMPLETED!`).
* При потере всех 3-х жизней от столкновения с врагами игра завершается экраном `GAME OVER`.

## 📁 Структура проекта

Основной код игры инкапсулирован в пакете `my_platformer/`:

* `main.py` — Точка входа, игровой цикл и обработка глобальных состояний.
* `config.py` — Константы конфигурации (разрешение окна `800x550`, FPS, цветовые палитры).
* `levels.py` — База данных уровней (координаты платформ, спавны врагов и предметов).
* `sprites.py` — Менеджер ресурсов: нарезка spritesheet-ов, загрузка слоев фона и анимаций.
* `menu.py` — Главное меню с динамическим параллаксом и логикой Hover-эффектов на кнопках.
* `pause_menu.py` — Игровое меню паузы и настройки.
* `player.py` — Физика игрока, обработка инпутов, логика прыжков и получения урона.
* `enemy.py` — Логика поведения врагов (патрулирование по заданным границам).
* `item.py` — Объекты сбора (монеты) и триггер перехода (портал).
* `world.py` — Управление игровым миром, генерация декораций, обработка коллизий и рендеринг.
* `utils.py` — Отрисовка сбалансированной HUD-панели статуса и загрузка текстур интерфейса.
* `test_game.py` — Скрипт автоматического тестирования в headless-режиме.
