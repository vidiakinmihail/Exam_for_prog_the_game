## Платформер для итогового проекта

Это рабочий прототип платформера на Pygame с логикой, коллизиями, камерой, предметами, врагами, здоровьем, переходами между уровнями, победой и поражением. Графика собрана из квадратов, прямоугольников и кругов, чтобы потом было удобно заменить её на спрайты.

### Запуск

Просто запусти:

```powershell
python main.py
```

### Управление

- `A` / `D` или `Left` / `Right` - движение
- `Space` - прыжок
- `Esc` - выход

### Состояния игры

- Уровень 1: собери 2 предмета, коснись двери и перейди на следующий уровень.
- Уровень 2: собери 4 предмета, коснись двери и перейди на следующий уровень.
- Уровень 3: собери 6 предметов, коснись двери и получи `GAME COMPLETED!`.
- Если жизни заканчиваются, появляется `GAME OVER`.

### Структура

Основная логика находится в папке `my_platformer/`:

- `main.py` - запуск игры
- `config.py` - все константы уровня и баланса
- `camera.py` - камера, следующая за игроком
- `player.py` - игрок, движение, прыжок, здоровье
- `enemy.py` - враг со state machine
- `item.py` - предметы и дверь
- `world.py` - мир, платформы, переходы между уровнями и проверки победы/поражения
- `utils.py` - HUD и текст

### Как добавить новый уровень

1. Открой [my_platformer/levels.py](my_platformer/levels.py).
2. Добавь новый ключ в `LEVELS`, например `4: { ... }`.
3. Заполни `platforms`, `items`, `enemies`, `door_spawn_platform` и `items_to_collect`.
4. Если нужен другой старт игрока или новая логика, можно подправить [my_platformer/world.py](my_platformer/world.py) или [my_platformer/config.py](my_platformer/config.py).
